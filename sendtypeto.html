sendtypeto - Particl RPC

sendtypeto "typein" "typeout" [{"address":"str","amount":amount,"narr":"str","blindingfactor":"hex","subfee":bool,"script":"hex","stakeaddress":"str"},...] ( "comment" "comment_to" ringsize inputs_per_sig test_fee {"changeaddress":"str","inputs":[{"tx":"hex","n":n},...],"replaceable":bool,"conf_target":n,"estimate_mode":"str","avoid_reuse":bool,"feeRate":amount,"blind_watchonly_visible":bool,"spend_frozen_blinded":bool,"test_mempool_accept":bool,"use_mixins":[ao_index,...],"mixin_selection_mode":n,"show_hex":bool,"show_fee":bool,"submit_tx":bool} )

Send part to multiple outputs.

Arguments:
1. typein                                  (string, required) part/blind/anon
2. typeout                                 (string, required) part/blind/anon
3. outputs                                 (json array, required) A json array of json objects
     [
       {                                   (json object, required)
         "address": "str",                 (string, optional, default=) The particl address to send to.
         "amount": amount,                 (numeric or string, optional, default=) The amount in PART to send. eg 0.1.
         "narr": "str",                    (string, optional, default=) Up to 24 character narration sent with the transaction.
         "blindingfactor": "hex",          (string, optional, default=) The blinding factor, 32 bytes and hex encoded.
         "subfee": bool,                   (boolean, optional, default=) The fee will be deducted from the amount being sent.
         "script": "hex",                  (string, optional, default=) Hex encoded script, will override the address. "address" must be set to a blank string or placeholder value when "script" is used. 
         "stakeaddress": "str",            (string, optional, default=) If set the output will be sent to a coldstaking script.
       },
       ...
     ]
4. comment                                 (string, optional, default=) A comment used to store what the transaction is for.
                                           This is not part of the transaction, just kept in your wallet.
5. comment_to                              (string, optional, default=) A comment to store the name of the person or organization
                                           to which you're sending the transaction. This is not part of the 
                                           transaction, just kept in your wallet.
6. ringsize                                (numeric, optional, default=5) Only applies when typein is anon.
7. inputs_per_sig                          (numeric, optional, default=1) Only applies when typein is anon.
8. test_fee                                (boolean, optional, default=false) Only return the fee it would cost to send, txn is discarded.
9. coin_control                            (json object, optional, default=)
     {
       "changeaddress": "str",             (string, optional, default=) The particl address to receive the change
       "inputs": [                         (json array, optional, default=) A json array of json objects
         {                                 (json object, optional, default=)
           "tx": "hex",                    (string, required) txn id
           "n": n,                         (numeric, required) txn vout
         },
         ...
       ],
       "replaceable": bool,                (boolean, optional, default=) Marks this transaction as BIP125 replaceable.
                                           Allows this transaction to be replaced by a transaction with higher fees
       "conf_target": n,                   (numeric, optional, default=) Confirmation target (in blocks)
       "estimate_mode": "str",             (string, optional, default=UNSET) The fee estimate mode, must be one of:
                                           "UNSET"
                                           "ECONOMICAL"
                                           "CONSERVATIVE"
       "avoid_reuse": bool,                (boolean, optional, default=true) (only available if avoid_reuse wallet flag is set) Avoid spending from dirty addresses; addresses are considered
                                           dirty if they have previously been used in a transaction.
       "feeRate": amount,                  (numeric or string, optional, default=not set: makes wallet determine the fee) Set a specific fee rate in PART/kB
       "blind_watchonly_visible": bool,    (boolean, optional, default=false) Reveal amounts of blinded outputs sent to stealth addresses to the scan_secret
       "spend_frozen_blinded": bool,       (boolean, optional, default=false) Enable spending frozen blinded outputs
       "test_mempool_accept": bool,        (boolean, optional, default=false) Test if transaction would be accepted to the mempool, return if not
       "use_mixins": [                     (json array, optional, default=) A json array of anonoutput indices to use as mixins
         ao_index,                         (numeric, optional, default=) anonoutput index
         ...
       ],
       "mixin_selection_mode": n,          (numeric, optional, default=) Mixin selection mode: 1 select from ranges, 2 select nearby, 3 random full range
       "show_hex": bool,                   (boolean, optional, default=false) Display the hex encoded tx
       "show_fee": bool,                   (boolean, optional, default=false) Return the fee
       "submit_tx": bool,                  (boolean, optional, default=true) Send the tx
     }

Result:
"txid"              (string) The transaction id.

Examples:
> particl-cli sendtypeto anon part "[{\"address\":\"PbpVcjgYatnkKgveaeqhkeQBFwjqR7jKBR\",\"amount\":0.1}]"

Particl version: 0.19.2.19
