rollbackrctindex - Particl RPC

rollbackrctindex

Roll back RCT index to current chain tip.

Result:
{
  "height" : int,                 (numeric) Current chain height.
}

Examples:
> particl-cli rollbackrctindex 
> curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "rollbackrctindex", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:51735/

Particl version: 0.19.2.19
